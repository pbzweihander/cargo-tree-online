# Cargo Tree Online

[Check out rendered page](https://pbzweihander.github.io/cargo-tree-online/)

Render cargo dependency tree in online.

## Usage

```
trunk serve
```

Copy and paste the content of `Cargo.lock` file to textarea and click submit button.
